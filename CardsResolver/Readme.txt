К сожалению, я не занимался раньше проблемами распознаванием образов. Так что, это мой первый опыт. 
Программа написана на Java без использования сторонних библиотек по распознаванию образов. 
В основе реализованного мной алгоритма лежит бинаризация изображений методом Оцу.
Распознавание осуществляется по рангу карт и по масти. Соответственно для этого,
были подготовлены файлы шаблонов с бинаризованными изображениями рангов и мастей.
Для работы с картинками использовалась программа paint.net
Для уточнений, в коде приведены комментарии в некоторых местах.
Для запуска программы, необходимо распаковать файл CardsResolver.rar в любое место и запустить
файл run.cmd <путь к папке с изображениями для распознавания>
Если <путь к папке с изображениями для распознавания> не указана, то поиск файлов *.png
осуществляется в текущей папке.
Программа упакована в jar в файле CardsResolver.jar
Исходники программы можно скачать с https://github.com/akartkam/CardsResolver.
Для удобства предполагается, что путь к файлу <jre_path>\bin\java.exe (для запуска программы)
прописан в переменной окружения path операционной системы.
Для запуска программы необходимы только 2 файла: CardsResolver.jar и run.cmd
