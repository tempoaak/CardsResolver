Программа для распознавания образов игральных карт из определенных унифицированных изображений определенного формата 
(карты находятся по центру изображения).
Для примера некоторые изображения находятся в папке SimpleImages.
Программа написана на Java без использования сторонних библиотек по распознаванию образов. 
В основе реализованного мной алгоритма лежит бинаризация изображений методом Оцу.
Распознавание осуществляется по рангу карт и по масти. Соответственно для этого,
были подготовлены файлы шаблонов с бинаризованными изображениями рангов и мастей.
Программа выводит в консоль информацию в формате <имя файла с изображением для распознавания> - <распознанные карты в мнемоническом вмде>
Мнемонический вывод распознанных карт состоит из двухсимвольных наборов и включает в себя ранг карты(цифру или первую букву названия ранга на английском, 
за которой следует название масти карты на английском). Например, вывод 5cQs6h , означает 5 треф дама пик шестерка черви
Для уточнений, в коде приведены комментарии в некоторых местах.
Для запуска в Windows, можно воспользоваться командой run.cmd <путь к папке с изображениями для распознавания>
(Либо можно напрямую java -jar CardsResolver.jar <путь к папке с изображениями для распознавания>)
Если <путь к папке с изображениями для распознавания> не указана, то поиск файлов *.png
осуществляется в текущей папке.
Программа упакована в jar в файле CardsResolver.jar
Исходники программы можно скачать с https://github.com/tempoaak/CardsResolver
Для удобства предполагается, что путь к файлу <jre_path>\bin\java.exe (для запуска программы)
прописан в переменной окружения path операционной системы.
